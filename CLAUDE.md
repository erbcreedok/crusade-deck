# Crusade Deck — контекст проекта

## Что это
Мультиплеерная карточная игра для мобильных браузеров. 2–32 игрока в сессии,
выбор колоды 36 или 52 карты. Сейчас в фазе фундаментальной инфраструктуры:
сначала connection/room architecture, потом уже игровые правила.

## Стек
- **Сервер**: Colyseus (Node.js)
- **Клиент**: React + Vite + Framer Motion
- **Auth**: Firebase Authentication (Google, Apple, email/password, magic link, anonymous guest)
- **Хостинг (план)**: self-hosted Colyseus на Railway или Fly.io
- **Возможно позже**: Redis для горизонтального масштабирования

## Что уже готово

### Сервер (Colyseus)
- `CardRoom` — основная игровая комната
- Схемы `GameState` / `Player`
- Верификация Firebase-токенов на сервере
- Реестр 6-значных инвайт-кодов для приватных комнат
- HTTP-эндпоинты
- Проходит `tsc --noEmit` без ошибок

### Клиент (React/Vite)
- Полные флоу авторизации: Google, Apple, email/password, magic link,
  анонимный гостевой вход, восстановление сессии
- Лобби: список публичных комнат + вход по коду
- Экран комнаты: готовность игроков, управление хостом
- Проходит `tsc --noEmit` без ошибок

### Есть также старый scaffold (вероятно устаревший)
Socket.io + Express + nanoid (`Room.js`, `RoomManager.js`, `server.js`).
Colyseus-версия — текущее направление, но в старом scaffold может быть
переиспользуемая логика (state machine комнаты, передача хоста при дисконнекте).

## Важные решения и почему
- Инфраструктура сознательно строится раньше игровой логики — это осознанный
  выбор последовательности, не забыли про правила игры, это следующий шаг.
- Латентность ~100–150ms приемлема для карточной пошаговой игры → удалённые,
  но производительные серверы (например Hetzner) ok, даже если физически
  далеко от Казахстана — если только не важна локальная юрисдикция.
- Framer Motion выбран специально под Balatro-style взаимодействие с картами
  (наклон при hover, физика drag со spring, веерная раскладка руки в hand) —
  не дефолт, а осознанный UX-выбор.
- Пользователь (я) — в Казахстане, это влияет на выбор хостинга:
  Hetzner CX22 — лучшее соотношение цена/производительность из доступных
  вариантов; Serverspace.kz — альтернатива, если важна локальная юрисдикция.
